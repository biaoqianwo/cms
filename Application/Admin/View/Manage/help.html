    概述

    本系统集成了内容管理系统(CMS)、企业官网、商城系统(Shop)等功能，根据您的业务不同，可以方便的选择

    1主题自定义

    2基于角色的权限系统,不同的管理员看到的菜单不同

    3前台导航条自定义

    4内容管理：文档内置了文章、帖子、活动、实体、专题、单页、友链7个列表模板，文章、帖子、活动、实体、单页5个详情模板，基本满足您对文档的全部需求

    5商城管理：基本满足您对商城的全部需求。且售后、报表一应俱全

    6期待更多的功能...

    基础设置

    设置是第一步：

    在您使用之前，已经为您内置了一些配置，如果您不熟悉本系统，就不用关心。如果您熟悉系统，可以自行修改：功能开关

    您需要设置的有网站设置 公司设置 还可以进行角色管理 管理员管理

    内容管理

    如果您的需求是内容管理系统CMS，这是您最常用的操作：

    1开始文档（包括文章）之前，您可以先进行分类管理 标签管理

    2当文档分类和标签管理好之后，就可以进行文档添加了。

    3还可以在文档行为里面看到活动的报名信息，以及用户喜欢文档的信息。

    商城管理

    如果您的需求是商城系统，这是您最常用的操作：

    1先进行分类管理，就可以进行商品添加

    2商城离不了订单管理 售后管理 售后分类

    3如果您的商品很多，且存在比较相近的，可以使用复制商品的功能。
